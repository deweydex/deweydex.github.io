Tutorial Task: How to Navigate a Messy Internet

See if you can find 3-5 different HTML/CSS tutorials online that we have not discussed in the past week and compare and contrast them.

Using Markdown cells in a Jupyter Notebook, address these questions:

  • How is the tutorial structured, what media are used? Is it a series of lessons? Projects? Videos?
  • What is your first impression of the tutorial site itself?
  • How long do you think it would take to do this tutorial?
  • How do you know if this resource is credible or up to date?
  • Is Javascript involved? How much knowledge is expected?
  • Would you personally use this resource?

Assessment

Assessment is project-based and continuous. You will be evaluated on:

Web Authoring Component

  • Creating a multi-page website using HTML, CSS, and JavaScript
  • Using GitHub for version control
  • Responsive design implementation
  • Code quality and organization
  • Documentation and commenting

Database Component

  • Understanding of SQL queries (SELECT, INSERT, UPDATE, DELETE)
  • Database design and normalization
  • Working with real-world datasets
  • Integration of database with web application
