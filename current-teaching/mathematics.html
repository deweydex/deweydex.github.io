Course Overview

This course explores mathematical concepts essential for IT and computer science. We'll be using Python and Jupyter notebooks to make mathematics more tangible and applicable to real-world problems. The emphasis is on understanding concepts deeply rather than memorization.

Course Plan for the Year

  • September: Review of foundations - Fractions, exponents, logarithms, graphing, algebra
  • October: Introduction to Functions and Matrices
  • November-December: Probability, Statistics, and Functions
  • January: Functions, Calculus (Basic Derivatives and Integrals), and plotting
  • February: Odds and Ends (Trigonometry and Tying things together)
  • March: Neural Networks Project with Digit and Letter Recognition, and Exam Review
  • April: Exam Review and Exam
  • May: Exam Results
Note: All work will be done using Python in Jupyter Notebooks. You'll learn to code mathematical concepts, which helps develop both mathematical and programming skills simultaneously.

Week of Oct 14-24: Matrix Multiplication

October 14-15: Transition to Matrix Multiplication

Today we are going to make the transition from the dot product to matrix multiplication (both on the board and in code).

Resources:

  • Start with Linear Algebra Fundamentals
  • Python and Jupyter tutorials available in notebooks

October 17: Matrix Multiplication Continued

Today we continue with Matrix Multiplication, thinking through a few key examples and shapes of matrices:

  • Diagonal matrices
  • Identity matrix
  • Zero matrix
  • Row multiplier
  • Row adder

We'll see how they work in the 2×2 and 3×3 case and begin to introduce how that looks visually. For fancier graphics, check out 3Blue1Brown's Series on Linear Algebra (both written and video)—we will be stopping at Cramer's Rule.

October 21: Matrix Exponentiation

Today we explored more examples of 2×2 matrix multiplication as well as taking a matrix to a given power (using exponents). We discussed the pattern that happens when we have the matrix [[1,1],[0,1]] and take it to the nth power.

In addition we went through the indexing that we use with matrices, how this "box of numbers" forces us to have a really clear idea of two indices combined. We also went through the "cross" of indices above, below, left and right, as well as how these relate to dimension.

October 22: Visual Representation

Today we went through some of the code for matrix multiplication as well as how things look visually (using the code in the linear algebra notebook). We will also discuss a couple examples of "interesting" 2×2 and 3×3 matrices like rotation matrices and scaling matrices to build up our "programming language of matrices".

October 24: List Comprehensions

Today we worked on the first 3 sections of the Lists and Matrices Notebook. We then discussed how to think about list comprehensions as for loops and how those relate to our matrix operations. Finally we went through the plan for Maths and assignments (in class skills demos) when you return.

November 4: Practice Problems and Square Root of -I

Summary of What We Have Covered So Far

  • Matrices as "Box-o-numbers"
  • Vectors as "List-o-numbers"
  • Dot product
  • Matrix multiplication
  • Exponents and matrix exponentiation (and some patterns that might arise)
  • Solving linear equations with matrices (three operations):
    • Swap Rows
    • Add one row to another row (leaving the first row alone)
    • Multiply a row by a scalar (gives us subtraction and division)
    • Any combination of the three until we get to the identity matrix on the left and a series of constants on the right

Practice Problem for Today

Can you find a 2×2 matrix that multiplies with itself to get -I (the Identity matrix (2×2) times negative 1)? What does that matrix look like? Let's call this the square root of -I or √(-I) = J. If you finish that, can you find something that squares to make J? Did you have to use a different method?

Assessment

Assessment in this course is continuous and skills-based. You will be evaluated on:

  • Skills Demonstrations (40%): In-class demonstrations of mathematical concepts and programming implementation
  • Portfolio/Collection of Work (60%): Jupyter notebooks, practice problems, and projects with 1:1 feedback

Key Skills to Demonstrate

  • Understanding of vectors and matrices
  • Ability to perform matrix operations by hand and in code
  • Solving linear equations using matrix methods
  • Creating visualizations of mathematical concepts
  • Applying mathematical concepts to programming problems
  • Building and understanding a simple neural network (final project)
Philosophy: In my courses, you can expect me to be fully transparent about my policies, assignments, and assessments. One of my main goals is that you become more aware of both how you learn and how to find and evaluate the information you need. Notes are welcome but not necessary—all information exists "out there" and I'll provide multiple resources with different looks, feels, and media.
